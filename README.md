# homebrew-cctools

Homebrew tap for Claude Code related tools.

## Installation

```bash
brew tap Nirvana-Jie/cctools
```

## Available Formulae

### claude-code-notifier

macOS system notifications for Claude Code - get notified when Claude needs your input.

```bash
brew install claude-code-notifier
```

**Features:**
- Receive notifications when Claude needs approval for file edits, bash commands, etc.
- Click notifications to return to your terminal
- Supports iTerm2, Terminal.app, VS Code, Warp, Hyper, Alacritty, and kitty

**Test installation:**
```bash
claude-code-notify
```

**More info:** [claude-code-notifier](https://github.com/Nirvana-Jie/claude-code-notifier)
